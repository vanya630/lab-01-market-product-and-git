## Roles and responsibilities

### Mobile Engineer (iOS/Android)
Designs, develops, and maintains native or cross-platform mobile applications. Collaborates with product and design teams to implement features, ensures performance and responsiveness, and handles deployment through app stores [[2]].

### Backend Engineer
Builds and maintains server-side logic, APIs, and database systems. Optimizes data flow between frontend and backend, implements security measures, and ensures scalability of services [[11]].

### DevOps Engineer
Manages CI/CD pipelines, automates infrastructure provisioning using IaC tools, monitors system health, and ensures rapid, reliable software delivery by bridging development and operations teams [[21]].

### QA Engineer
Creates and executes test plans (manual and automated), identifies bugs across development stages, validates fixes, and ensures the final product meets functional and non-functional requirements before release [[31]].

### Data Engineer
Designs and maintains data pipelines for ingestion, transformation, and storage. Ensures data quality, builds data lakes/warehouses, and enables analytics and ML workflows through robust, scalable architectures [[41]].

- Strong problem-solving and debugging abilities
